Developer Guide

Всё, что нужно для создания и публикации игры на PlayRU Platform.

Quick Start

  1. Зарегистрируйтесь как разработчик на /dev/register/
  2. Создайте игру — перейдите в Новая игра и заполните форму
  3. Укажите репозиторий — ссылка на Godot-проект (GitHub/GitLab)
  4. Отправьте на модерацию — нажмите "Отправить на модерацию"
  5. После одобрения игра автоматически появится в каталоге для всех игроков

Структура Godot-проекта

games/my_game/
├── MyGame.gd          # Главный скрипт (точка входа)
├── MyGame.tscn        # Главная сцена
├── game.cfg           # Метаданные игры (название, версия)
└── assets/            # Ресурсы (спрайты, звуки)

game.cfg (пример)

[game]
title = "Моя Игра"
slug = "my-game"
version = "1.0.0"
genre = "arcade"
min_age = 6
min_players = 1
max_players = 1

[entry]
scene = "MyGame.tscn"

Контракт с платформой

Ваша игра взаимодействует с платформой через GameManager:

# Завершить игру и вернуться в лобби
GameManager.end_game()

# Отправить результат (score)
GameManager.submit_score(score: int)

# Получить данные текущего игрока
var player = GameManager.get_current_player()
# player.nakama_user_id, player.display_name

Platform API Reference

Base URL: https://api.playru.ru/api/v1

Игровые сессии

POST /sessions/start/

Начало игровой сессии. Вызывается при запуске игры.

Request:  {"nakama_user_id": "abc123", "game_slug": "my-game", "platform": "android"}
Response: {"session_id": "uuid", "started_at": "2026-03-09T12:00:00Z"}
POST /sessions/{session_id}/end/

Завершение сессии с результатом.

Request:  {"score": 1500, "completed": true, "duration_seconds": 120}
Response: {"session_id": "uuid", "score": 1500, "ended_at": "..."}

Лидерборд

GET /games/{slug}/leaderboard/?period=all|week|day

Топ-50 игроков по лучшему скору.

Response: {
  "game_slug": "my-game",
  "period": "all",
  "total_players": 3,
  "records": [
    {"rank": 1, "display_name": "Player1", "score": 5000},
    {"rank": 2, "display_name": "Player2", "score": 3200}
  ]
}

Профиль игрока

POST /profile/sync/

Создать/обновить профиль игрока.

Request:  {"nakama_user_id": "abc123", "display_name": "Player1", "auth_provider": "vk"}
Response: {"nakama_user_id": "abc123", "display_name": "Player1", "created": true}
GET /profile/{nakama_user_id}/

Получить профиль игрока по Nakama ID.

Рейтинг и отзывы

POST /games/{slug}/rate/

Поставить оценку игре (1-5 звёзд).

Request:  {"nakama_user_id": "abc123", "stars": 5, "review": "Отличная игра!"}
Response: {"success": true, "game_avg_rating": 4.5, "game_ratings_count": 12}
GET /games/{slug}/reviews/

Список одобренных отзывов (последние 20).

Аналитика

POST /games/{slug}/play/

Засчитать запуск игры (инкремент play_count).

Требования и ограничения

Технические

  • Godot 4.x
  • Разрешение: 1280x720 (landscape) или 720x1280 (portrait)
  • Максимальный размер .pck: 50 МБ
  • Запрещено: доступ к файловой системе, сеть напрямую (только через API)

Контентные

  • Возрастной рейтинг: от 6+
  • Без насилия, 18+ контента
  • Без рекламы сторонних сервисов
  • Русский язык интерфейса обязателен

Жизненный цикл публикации

Черновик → На модерации → Опубликована

Если игра отклонена, вы получите причину отказа и сможете исправить замечания и отправить повторно.
